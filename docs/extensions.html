EXTENSIONS

MathUtils:
use(__mathutils)
Imports the MathUtils extension, for doing more advanced math operations.
run(/__mathutils:square[num] => destination)
Squares a number and stores the result in the specified destination.
run(/__mathutils:cube[num] => destination)
Cubes a number and stores the result in the specified destination.
run(/__mathutils:pow[num, pow] => destination)
Raises a number to the specified power.
run(/__mathutils:abs[num] => destination)
Returns the absolute value of a number.
run(/__mathutils:min[num1, num2] => destination)
Returns the smallest of the two numbers.
run(/__mathutils:max[num1, num2] => destination)
Returns the largest of the two numbers.
3Ops:
use(__3ops)
Imports the 3Ops extension, which provides basic arithmetic operations for three numbers.
run(/__3ops:add3[num1, num2, num3] => destination)
Adds three numbers and stores the result in the specified destination.
run(/__3ops:sub3[num1, num2, num3] => destination)
Subtracts the three numbers (Left to right) and stores the result in the specified destination.
run(/__3ops:mul3[num1, num2, num3] => destination)
Multiplies the three numbers and stores the result in the specified destination.
run(/__3ops:div3[num1, num2, num3] => destination)
Divides the three numbers (Left to right) and stores the result in the specified destination.
ListUtils:
use(__listutils)
Imports the ListUtils extension, which provides additional functions for working with lists.
run(/__listutils:sort[list] => destination)
Sorts the list literal and stores the result in the specified destination. Lists with numbers are sorted least => greatest and string lists are sorted alphabetically. For a list with mixed types, numbers always come before the strings.
run(/__listutils:count[list, element] => destination)
Counts the amount of instances of an element in the list and stores the result in the specified destination.
run(/__listutils:unique[list] => destination)
Returns a new list with all duplicate elements removed. (List order is mostly preserved)
StringUtils:
use(__stringutils)
Imports the StringUtils extension, which provides additional functions for working with strings.
run(/__stringutils:separate[str, delimiter] => destination)
Splits a string into a list of substrings based on the specified delimiter. Note: The delimiter has to be one single character.
run(/__stringutils:join[list, delimiter] => destination)
Joins a list of strings into a single string, using the specified delimiter.
run(/__stringutils:isempty[str] => destination)
Returns 'true' if a string is empty, and 'false' if not.
